SF2-Suite RT

Die produktionsnahe Real-Time Edition läuft lokal auf Ihrer Workstation und verarbeitet Sensordaten in Echtzeit über Netzwerk-Streaming, inklusive SCADA-Übergabe und Deployment.

Was Suite RT ergänzt

Suite RT enthält die vollständige Suite-SE-Funktionalität und erweitert den Stack um produktionsrelevante Echtzeit- und Rollout-Fähigkeiten.

  • Verarbeitung von Live-Datenströmen aus Sensor-Gateways
  • File Stream Server für realitätsnahe Stream-Simulation
  • SCADA-Simulator zur Validierung ohne reale Anlage
  • Kompilierung eines vollständigen Produktionspakets

Warum für Applikationsingenieure?

Die Suite schließt die Lücke zwischen lokaler Evaluierung und stabilem Produktionsbetrieb, ohne Tool-Wechsel und ohne Cloud-Abhängigkeit.

  • Schneller Transfer vom PoV in die echte Anlage
  • Reproduzierbare Deployments über eindeutige Pakete
  • Zentrale Sicht auf mehrere Runtime-Instanzen
  • Direkte Trigger-Ausgabe für bestehende OT-Landschaften

Integrationspfad in 4 Schritten

1. Modell aus Suite SE übernehmen

Use-Case-Logik und Fingerprint-Bibliotheken aus der Engineering-Phase werden unverändert in die produktive Pipeline überführt.

2. Live-Input anbinden

Entweder direkt aus Gateway-Daten oder über emulierte Streams aus dem File Stream Server für reproduzierbare Tests.

3. Triggerkanäle validieren

SCADA-Simulator und reale Ausgabepfade erlauben das Feintuning von Schwellwerten, Alarmen und Zustandsübergängen.

4. Runtime-Paket ausrollen

Kompiliertes Paket wird als SF2-Engine betrieben und über die Suite zentral überwacht und versioniert.
