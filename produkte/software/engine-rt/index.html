Software / Runtime Module

SF2-Engine

Das kompakte Runtime-Modul für den robusten Produktionsbetrieb: ausgerollte Modelle ohne Benutzeroberfläche ausführen, deterministisch und lokal direkt an der Anlage.

Headless Runtime Docker Image SF2 Hardware-Modul Low-Latency
Runtime-Lizenz anfragen Zur Suite RT

Runtime ohne Umwege

Engine übernimmt exakt das in Suite SE/Suite RT definierte Verhaltensmodell und führt dieses im Feld ohne UI und ohne Modellabweichung aus.

  • Ausführung fertig konfigurierter Produktionspakete
  • Deterministische Fingerprint-Logik im laufenden Betrieb
  • Stabile Triggerkanäle für SCADA/OT-Ausgabe
  • Betrieb in abgeschotteten Netzwerken möglich

Deployment-Optionen

Engine kann softwarezentriert oder hardwaregestützt betrieben werden, abhängig von Ihrer OT-Infrastruktur.

  • Docker-Image für containerisierte Deployment-Umgebungen
  • Native Ausführung auf dem SF2-Edge Device
  • Zentrale Verwaltung der Instanzen über Suite RT
  • Versionierte Rollouts über reproduzierbare Pakete
