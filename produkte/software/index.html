Software / Übersicht

Der SF2 Software-Stack

Suite SE und Suite RT laufen beide lokal auf der Workstation: Suite SE arbeitet ausschließlich mit Sensor-Logs aus Dateien, Suite RT verarbeitet zusätzlich Sensordaten in Echtzeit über das Netzwerk.

No-Cloud Deterministisch CSV bis Live-Stream Self-Service-first

SF2-Suite SE

Die freie SE Edition für den risikofreien Einstieg: lokal auf der eigenen Workstation und ausschließlich mit Sensor-Logs aus Dateien (CSV).

  • CSV-basierte Logfile-Verarbeitung
  • UDTAT/UDM/UFP Workflow (Universal Data Transformation and Analytic Tool, Universal Data Mapper, Universal Finger Printer)
  • Interaktive Fingerprint-Bildung
Produktdetails

SF2-Suite RT

Die Produktionssuite für lokale Workstations mit zusätzlicher Echtzeitverarbeitung über Netzwerk-Streams, SCADA-Integration und Flottenmanagement.

  • Live-Datenquellen und Simulatoren
  • SCADA-Ausgabe in Echtzeit
  • Zentrale Verwaltung ausgerollter Engines
Produktdetails

SF2-Engine

Das headless Runtime-Modul für den robusten Betrieb ausgerollter Modelle in Produktionsumgebungen.

  • Docker-Image oder SF2-Hardware-Modul
  • Deterministische Laufzeitlogik
  • Optimiert für Edge-Betrieb
Produktdetails

Vergleichsmatrix: SF2-Suite SE vs. SF2-Suite RT

Funktion SF2-Suite SE SF2-Suite RT
Lokaler Betrieb ohne Cloud-Backend Ja Ja
Ausschließlicher File-Log-Modus (Sensor-Logs aus Dateien) Ja Nein
UDTAT / UDM / UFP Workflow (Universal Data Transformation and Analytic Tool / Universal Data Mapper / Universal Finger Printer) Ja Ja
Interaktive Visualisierung und Fingerprint-Bildung Ja Ja
Echtzeit-Streams von Sensor-Gateways Nein Ja
File Stream Server zur realistischen Simulation Nein Ja
Weitergabe von Triggern an SCADA Nein Ja
Kompilierung eines deploybaren Produktionspakets Nein Ja
Flottenmanagement mehrerer Runtime-Instanzen Nein Ja
Self-Service-Einstieg über kostenfreien Lizenzschlüssel Ja Nein

Empfohlener Pfad: Einstieg mit Suite SE für schnelle Use-Case-Erprobung, anschließend produktiver Rollout mit Suite RT.

Konversionspfad für Engineering-Teams

Phase 2: Produktion

Suite RT und Engine übernehmen den Übergang in Echtzeitbetrieb, Rollout und skalierbares Runtime-Management.
