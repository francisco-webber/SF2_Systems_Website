# SF2 Systems — Statische Website

## Inhalt
- `index.html` — Landing Page mit Workflow-, Trust- und Roadmap-Abschnitten
- `produkte/` — Software- und Hardware-Produktseiten mit technischen Eckdaten
- `shop/` — Download-/Lizenzpfade und Angebotsübersicht
- `support-manuals/` — Quickstart, Integrationspfad und Proof Center
- `downloads/brochures/` — Produktunterlagen als PDF-Download (z. B. Product Folder)
- `downloads/studio-se/` — Starter-Kit, Demo-CSV und Plattformleitfäden
- `assets/images/` — Medienstruktur, Platzhalter und Screenshot-Pattern (`README.md`)
- `assets/` — gemeinsames Stylesheet, JavaScript und Logo

## Start
1. ZIP-Datei entpacken
2. `index.html` im Browser öffnen

Für lokales Testen mit sauberem Routing kann alternativ ein einfacher lokaler Webserver verwendet werden, z. B.:
- Python: `python3 -m http.server`
- dann `http://localhost:8000/` aufrufen
