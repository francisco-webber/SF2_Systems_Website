SF2 Studio SE - MacOSX-ARM64 Installationsleitfaden

1) Lizenzschluessel anfordern (berufliche E-Mail)
2) MacOSX-ARM64 Paket aus dem License Server herunterladen
3) App in den Programme-Ordner kopieren
4) Beim Erststart ggf. Sicherheitsfreigabe in den Systemeinstellungen erlauben
5) Studio starten und Lizenzschluessel hinterlegen

Erstvalidierung:
- demo-sensor-log.csv laden
- UDTAT-Konfiguration (Universal Data Transformation and Analytic Tool) anlegen
- Fingerprints fuer OPERATIONAL und ALERT pruefen
